# Mini-Ramses Analysis Tools 

This repository contains a collection of tools for analyzing and visualizing data generated by the Mini-Ramses simulation code. The tools are designed to facilitate the interpretation of simulation results. 

The current codes that are available in this repository include:

`miniramses.py`: A general workhorse for loading in mini-ramses data directly from snapshots. You can get `info`, `particle`, `grid`, and `clump` data from the snapshots. 
`mk_tracks.py`: A tool for generating evolutionary tracks of tree particles in the mini-ramses simulation. It can be used to visualize the evolution of particles over time.
`mergertree.py`: A tool for making mergertrees from the mini-ramses simulation data. One needs to run a simulation with `merger_tree=.true.` in the namelist to make evolutionary tracks with `mk_tracks.py`. This code uses the outputs of `mk_tracks.py` to generate the merger trees.

You can find minimal documentation in the `docs` directory with examples of how to use the tools.
